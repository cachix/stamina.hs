## Design goals

- Defaults that make sense most of the time.
- Simple interface.
- Documented.

Heavily inspired by [stamina for Python](https://stamina.hynek.me/en/stable/tutorial.html#retries).

## Development

1. Install [devenv.sh](https://devenv.sh/getting-started/).

2. `devenv shell`

3. `stack build`
